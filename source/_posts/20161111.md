## start oschina git
```
//第一步
git clone http://git.oschina.net/xxxxxx/xxxxxx.git

//创建分支,可以直接在界面创建
git checkout -b $feature_name

//第二步，进入git目录下面
git add .

git commit -am "My feature is ready"

git push origin $feature_name

```
