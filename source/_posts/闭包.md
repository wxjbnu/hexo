## 闭包

> JavaScript closures

- https://segmentfault.com/a/1190000005167435
- https://segmentfault.com/a/1190000006193111
- http://kb.cnblogs.com/page/110782/





#### 案例
```javascript
//error
var a = [];
for (var i = 0; i < 10; i++) {
   a[i] = function () {
       console.log(i);
   };
}
a[6]();

//闭包
var a = [];
for (var i = 0; i < 10; i++) {
    (function(i){
        a[i] = function () {
            console.log(i);
        };
    })(i)
}
a[6]();


```
